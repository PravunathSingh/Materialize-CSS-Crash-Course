This section is all about displaying and hiding content based on screen sizes. To hide any element, we need to add the "hide" class. And then we can add classes like:

  • hide-on-small-only
  • hide-on-med-only
  • hide-on-large-only
  • hide-on-med-and-down
  • hide-on-med-and-up
The above classes are self explanatory.
